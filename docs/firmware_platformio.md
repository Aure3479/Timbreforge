# Firmware (PlatformIO)

The current board firmware is stored in:
- `firmware/vibraforge_platformio/platformio.ini`
- `firmware/vibraforge_platformio/src/main.cpp`

!!! warning "Baseline"
    Firmware is based on Vibraforge and is normally used as-is.
    Only apply small edits if needed (e.g., BLE name/UUID).
